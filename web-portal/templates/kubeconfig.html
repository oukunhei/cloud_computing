Quick Usage Guide
  1. Download the kubeconfig for your namespace and role.
  2. Set the environment variable: export KUBECONFIG=./team-alpha-dev-kubeconfig.yaml
  3. Verify access: kubectl get pods
  4. Test permissions: kubectl get resourcequota (readable by all roles; modification requires admin)
